# Marketing Studio — Product Reference Documents

FY26 Q4 Release Planning · UX & PM Documentation Suite

## 📄 Documents

### 1. [IA Views Matrix](ia-views-matrix-updated.html)
Complete information architecture for Marketing Studio — **147 interactive rows** mapping every screen, view, and interaction pattern. Covers 60 entities across Skills, Artifacts, Assets, AI Interactions, Logs, and Cues. Each row defines view type, interaction modes, role context, info-vis variables, manipulation variables, and action variables. Includes **27 new rows** from PRFAQ gap analysis.

### 2. [Use Case Task Flows](ms-use-case-task-flows.html)
Sequential task breakdowns for all **22 use cases** across R1/R2/R3/Backlog — **375 tasks** with specific user roles, detailed info-vis needs (2,274 items), manipulation variables (1,714), actions (1,615), view type specs, and PM/UX guidance. The primary reference for building the product.

### 3. [AI Automation Opportunities](ms-ai-automation-opportunities.html)
Analysis of **20 AI capabilities** across 396 task instances. 16 are new suggestions not in the PRFAQ. 17 are production-ready today. Covers agentic AI, generative AI, pattern recognition, and MCP skill composition opportunities with impact assessment and UX implementation notes.
